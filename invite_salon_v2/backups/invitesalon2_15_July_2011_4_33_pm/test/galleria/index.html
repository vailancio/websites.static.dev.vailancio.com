Galleria what?

Galleria is a JavaScript image gallery framework built on top of the jQuery library. The aim is to simplify the process of creating professional image galleries for the web and mobile devices.

Get started →

One core, multiple themes

Galleria comes bundled with one free theme called ”Classic”. If you need something more, build your own using our API or try one of our Premium Themes, starting at only $9 per license.

Browse Themes →

Open Source & MIT licenced

Galleria is free to use and licensed under MIT, which means it places almost no restrictions in what way you want to use it. All the code along with the documentation is available in our github repositiory.
